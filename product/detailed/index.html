Tech Specs

  • SoC

    RockChip RK3566

  • CPU

    Quad‑core Arm® Cortex®‑A55 (ARMv8) 64‑bit @ 1.6GHz

  • GPU

    Arm Mali™‑G52‑2EE, OpenGL® ES1.1/2.0/3.2, Vulkan® 1.1, OpenCL™ 2.0

  • NPU

    1TOPs, support INT8/INT16.

  • RAM

    32-bit LPDDR4 RAM options:

    - 1GB

    - 2GB

  • USB

    2x USB2 HOST ports

    1x USB2 OTG/HOST port

    1x USB3 HOST port

  • Storage

    eMMC module: up to 512G eMMC module

    SD Card: up to 512G SD cards

    NVMe SSD: Support M.2 M Key 2230 SSD

  • Wireless

    802.11 b/g/n/ac Wireless LAN supporting WiFi 5

    BT 5.0 with BLE support

    On board antenna or external antenna options available

  • Ethernet

    1x Gigabit Ethernet port. It supports PoE with add-on PoE HAT

  • Display

    1x HDMI 2.0 port supporting displays up to 1080P@60fps resolution

    1x display port supporting 2-lane MIPI DSI

  • Camera

    1x 2-lane MIPI CSI camera connector

  • Audio

    3.5mm jack with microphone. The HD codec supports up to 24-bit/96KHz audio

  • Multimedia

    H.265/H.264 decoder up to 4K@60fps

    H.264/H.265 encoder up to 1080P@60fps

  • Dimension

    85 mm x 56 mm

  • Power

    USB Type‑C® 5V

    5V Power from the GPIO PIN 2 & 4
