Tech Specs

  • SoC

    Rockchip RK3566

  • CPU

    Quad‑core Arm® Cortex®‑A55 (ARMv8) 64‑bit @ 1.6GHz

  • GPU

    Arm Mali™‑G52‑2EE

    OpenGL® ES1.1/2.0/3.2, Vulkan® 1.1, OpenCL™ 2.0

    Embedded high performance 2D image acceleration module

  • RAM

    1GB / 2GB LPDDR4 available

  • Storage

    Storage is supported by

    eMMC storage through the eMMC connector

    Micro SD card through the Micro SD slot

    SSD through the M.2 connector

    SATA through the Radxa Penta SATA HAT

  • Audio

    1x 4‐ring 3.5mm headphone jack with mic input

  • Display

    Display via HDMI or MIPI DSI. They can not work at the same time

  • Video Input

    1x camera port supporting 2‑lane MIPI CSI

  • Multimedia

    H.264/H.265 decoder up to 4K@60fps

    H.264/H.265 encoder up to 1080P@60fps

  • USB

    2x USB2 HOST ports

    1x USB2 OTG/HOST port

    1x USB3 HOST port

  • Ethernet

    1x Gigabit Ethernet port(supports PoE with add-on PoE HAT)

  • Wireless

    802.11 b/g/n/ac Wireless LAN supporting WiFi 5

    BT 5.0

  • Miscellaneous

    40x user GPIO supporting various interface options:

    up to 5 x UART

    1 x SPI bus

    up to 2 x I2C bus

    1 x PCM/I2S

    up to 6 x PWM

    up to 28 x GPIO

    2 x 5V DC power in

    2 x 3.3V power pin

  • Power

    USB Type‑C® 5V

    5V Power from the GPIO PIN 2 & 4

  • Dimension

    87 mm x 55 mm
